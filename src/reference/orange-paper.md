# Orange Paper

The normative mathematical specification of Bitcoin consensus is published on the **Bitcoin Commons** website. This book does **not** embed the full Orange Paper text.

**Read the Orange Paper:** [thebitcoincommons.org/protocol.html](https://thebitcoincommons.org/protocol.html)

**Source & changes:** [blvm-spec](https://github.com/BTCDecoded/blvm-spec) on GitHub (Layer 1 constitutional repo).

**In BLVM:** [blvm-consensus](../consensus/overview.md) implements these rules; [BLVM Specification Lock](../consensus/formal-verification.md), tests, and review validate code against the spec. The Orange Paper is the IR—the implementation is not generated from it. See [compiler-like architecture](glossary.md#compiler-like-architecture).
