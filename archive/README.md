# Documentation Archive

This directory contains archived historical documentation from all BLLVM repositories.

## Structure

- `bllvm-consensus/` - Historical docs from bllvm-consensus (33 files)
- `bllvm-node/` - Historical docs from bllvm-node (7 files)
- `bllvm-commons/` - Historical docs from bllvm-commons (47 files)

## Purpose

These documents are historical snapshots preserved for reference. They document:
- Completed implementation phases
- Milestone achievements
- Status updates at specific points in time
- Migration plans and summaries

**Note**: These documents are no longer authoritative. For current documentation, see the individual repository README files and their active documentation directories.
